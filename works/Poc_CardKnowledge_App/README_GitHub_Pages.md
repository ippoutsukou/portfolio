# 新人向け業務ナレッジ PoC

GitHub Pagesで公開できる、静的HTMLのPoCです。

## ファイル
- `index.html`

## できること
- 質問カテゴリを選ぶ
- 商品 / 顧客 / 状況を入力する
- 条件分岐で回答を表示する
- 回答を「結論 / 適用条件 / 出典 / 次アクション」で固定表示する
- 価格・契約・納期確約などは禁止領域として自動回答しない

## GitHub Pages公開手順
1. GitHubで新しいリポジトリを作る
2. この `index.html` をアップロードする
3. リポジトリの `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. Branchを `main`、Folderを `/root` にする
6. 保存する
7. 数分待つと公開URLが発行される

## 補足
このPoCは「AIの賢さ」ではなく、「カード型ナレッジで安全に返す設計」を見せるためのものです。
