PriceFlow

価格を自動で出すだけでなく、「なぜその価格になるのか」を説明できるようにした Rule-Based Pricing PoCです。工程、資材、条件分岐、利益率を分解して見せることで、 判断根拠を共有しやすい構成にしています。

Static HTML Rule-Based Pricing Explainable Costing

概要

  • 製品種別、数量、仕様、サイズ、顧客区分などの条件から売価を試算する PoC です。
  • 計算結果だけでなく、材料費、工程費、間接費、利益率の積み上げを同時に表示します。

できること

  • Quick Presets で代表パターンを即時反映し、比較検討の出発点をそろえられます。
  • 入力値を変えると KPI と計算内訳が更新され、価格変動の理由を追いやすくしています。
  • 結果要約をコピーして、そのまま説明メモや検討ログに転記できます。

見どころ

  • 「自動化」よりも「説明可能性」を優先し、価格判断の透明性を高める設計にしています。
  • 単一 HTML で完結するため、ローカル確認や GitHub Pages 配置にそのまま使えます。
