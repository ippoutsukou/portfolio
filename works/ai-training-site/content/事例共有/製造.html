AIによる品質検査の自動化

課題

従来の目視検査では、検査員の疲労や集中力の低下により、品質のばらつきが発生していました。

解決策

ディープラーニングを活用した画像認識システムを導入し、製品の外観検査を自動化。24時間安定した品質検査を実現しました。

効果

  • 検査精度の向上（99.9%）
  • 検査時間の50%削減
  • 人件費の30%削減

使い方

検査用カメラで撮影した製品画像をAIに渡し、合否・欠陥種類・位置情報を自動出力します。結果はダッシュボードに表示し、CSV/Excelへ自動記録できます。

参考プロンプト例

# 役割
あなたは製造業の「外観検査アナリスト」です。欠陥の種類・位置・重症度を判定し、最終合否を出します。

# 入力
- 画像：製品の表面画像（1〜数枚）
- 仕様：良品基準（例：スクラッチ長 3mm 未満は許容 等）

# 出力フォーマット（JSON）
{{
  "result": "良品|不良品",
  "defects": [
    {{"type":"scratch","severity":"low|mid|high","position":"(x,y,w,h)","note":"任意"}}
  ],
  "reason":"判定理由を簡潔に",
  "actions":"再検査/現場指示/設備点検 など"
}}

# 制約
- 仕様に沿って厳密に判定してください。
- 曖昧な場合は「要再検査」としてください。

予知保全による設備故障の防止

課題

設備の故障による突発的な生産停止が、生産効率とコストに大きな影響を与えていました。

解決策

センサーデータと機械学習を組み合わせ、設備の異常を事前に検知する予知保全システムを導入。

効果

  • 突発的な設備故障の80%削減
  • メンテナンスコストの40%削減
  • 設備稼働率の15%向上

使い方

稼働データ（振動・温度・電流・音）を日次/時間単位で取り込み、しきい値逸脱や劣化傾向を自動判定。異常予兆はメール/Teamsへ通知、保全指示票を自動作成します。

参考プロンプト例

# 役割
あなたは設備診断エンジニア。センサーログから異常兆候を判定します。

# 入力
- CSV：timestamp, vibration, temp, current, sound
- 保全履歴：直近3か月の作業記録

# 期待する出力
1) 異常度スコア（0-100）と根拠（特徴量/傾向）
2) 故障予測（いつ・何が起きそうか）
3) 推奨アクション（点検項目/優先度/停止要否）
4) ダッシュボードに載せるサマリ（50字程度）

AIによる生産計画の最適化

課題

複雑な生産計画の立案に時間がかかり、在庫の過不足が発生していました。

解決策

需要予測と在庫最適化のAIシステムを導入し、リアルタイムでの生産計画の調整を実現。

効果

  • 在庫回転率の25%向上
  • 計画立案時間の70%削減
  • 納期遵守率の95%達成

使い方

販売実績・受注・在庫・能力・停止予定などのデータを取り込み、週次で自動計画を生成。変更点だけを人が最終確認して確定し、現場に自動配信します。

参考プロンプト例

# 目的
需要・在庫・能力制約を満たしつつ総コスト最小の週次生産計画を提案せよ。

# 入力
- 需要予測CSV（SKU/週/数量）
- 在庫水準（安全在庫含む）
- 能力制約（ライン別・シフト別・外注枠）
- 設備停止予定・段取り替え時間

# 出力
- SKU×週の生産数量案（CSV想定）
- 在庫推移/欠品リスクの可視化ポイント
- 制約に抵触する場合の代替案（加班/外注/前倒し等）
- 管理者向けサマリ（100字以内）
