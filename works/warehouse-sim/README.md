# 格納SIM 使い方

`warehouse-sim copy.html` は、倉庫内の荷物配置をドラッグ操作で作成し、床面積・占有率・収容数を確認できる単体HTMLアプリです。

## 起動方法

1. `warehouse-sim copy.html` をブラウザで開きます。
2. 追加のインストールやサーバー起動は不要です。
3. 保存・読込はブラウザのファイルダウンロード/アップロード機能を使います。

## 画面構成

- **倉庫設定**: 倉庫名、床の幅、奥行、グリッド間隔、格子表示、荷物吸着を設定します。
- **荷物プリセット**: 荷物・柱・通路の部品一覧です。プリセットを床へドラッグして配置します。
- **選択中の荷物**: 選択した荷物の段積み数、座標、回転、複製、削除、整列を操作します。
- **収容集計**: 設置数、総収容数、使用床面積、空き有効床、床占有率をリアルタイム表示します。
- **Before / After**: 現在状態をBefore保存し、変更後との差分を確認します。
- **記録一覧**: 複数の倉庫レイアウトを一覧管理し、CSV/JSONで出力できます。

## 基本操作

### 倉庫サイズを設定する

左側の **倉庫設定** で以下を入力します。

- **倉庫名**: 記録一覧に保存するときの名前です。
- **幅 W (m)**: 倉庫の横幅です。
- **奥行 D (m)**: 倉庫の奥行です。
- **グリッド (m)**: 床に表示する格子の間隔です。

`格子` をオフにするとグリッド表示を非表示にできます。`荷物吸着` をオンにすると、配置時に荷物同士の辺へ吸着しやすくなります。

### 荷物を配置する

1. 左側の **荷物プリセット** から使いたい荷物を選びます。
2. 床エリアへドラッグします。
3. 配置済みの荷物はドラッグで移動できます。
4. 荷物をクリックすると選択状態になります。

初期プリセットは以下です。

| プリセット | サイズ | 最大段積み |
| --- | --- | --- |
| パレット | 1.1 × 1.1m | 4段 |
| 大型 | 2.0 × 1.2m | 3段 |
| 小型 | 0.6 × 0.4m | 8段 |
| 長物 | 3.0 × 0.5m | 2段 |
| ネステナー | 1.2 × 1.1m | 4段 |

柱と通路は障害物として扱われ、使用床面積ではなく有効床面積から差し引かれます。

### プリセットを追加する

荷物プリセット欄の入力フォームに、名称、幅、奥行、段数、種別を入力して **+ プリセット追加** を押します。

種別は以下から選べます。

- **荷物（段積み可）**: 収容数と使用床面積の集計対象です。
- **柱**: 障害物として扱います。
- **通路**: 障害物として扱います。

プリセット右側の `×` を押すと、そのプリセットを一覧から削除できます。

## 選択中の荷物を編集する

荷物を選択すると、左側の **選択中の荷物** で編集できます。

- **段積み数**: `＋` / `−` で段数を変更します。最大段数はプリセットごとに決まります。
- **X座標 / Y座標**: メートル単位で位置を直接入力します。
- **コピー**: 選択中の荷物をコピーします。
- **90°回転**: 幅と奥行を入れ替えます。
- **複製**: 選択中の荷物を近くに複製します。
- **削除**: 選択中の荷物を削除します。

複数選択している場合は、左揃え、中央揃え、右揃え、上揃え、下揃え、横分配、縦分配が使えます。分配は3個以上を選択しているときに有効です。

## 格子複製

同じ荷物をまとめて並べたい場合は、荷物を選択して **格子複製** を使います。

1. 複製したい荷物を選択します。
2. 列数、行数、列間隔、行間隔を入力します。
3. **格子複製を実行** を押します。

床の外にはみ出す区画はスキップされます。

## Before / After 比較

1. 比較の基準にしたい状態で、上部の **Before** を押します。
2. その後、荷物の追加・移動・削除などを行います。
3. 左下の **Before / After** に、設置数、総収容数、使用床面積、床占有率の差分が表示されます。

`Before配置を重ねて表示` をオンにすると、Before時点の配置を半透明で重ねて確認できます。

## 保存と読込

### レイアウトを保存する

上部の **保存** を押すと、現在の倉庫レイアウトをJSONファイルとして保存します。

保存内容には以下が含まれます。

- 倉庫名
- 床サイズ
- グリッド設定
- プリセット
- 配置済みアイテム
- Before情報
- 集計サマリー

### レイアウトを読み込む

上部の **読込** を押して、保存済みのJSONファイルを選択します。読み込むと現在の画面がJSONの内容に置き換わります。

## 記録一覧

複数の倉庫レイアウトを比較・管理したい場合は、上部の **一覧に記録** を使います。

1. 倉庫名を入力します。
2. レイアウトを作成します。
3. **一覧に記録** を押します。
4. **一覧** を押すと、記録済みの倉庫が一覧表示されます。

同じ倉庫名で記録すると、既存の記録が更新されます。

一覧画面では以下ができます。

- **開く**: 記録したレイアウトをエディタへ読み込みます。
- **削除**: 記録を一覧から削除します。
- **CSV出力**: 集計一覧をCSVで保存します。
- **一覧JSON保存**: 記録一覧全体をJSONで保存します。
- **一覧JSON読込**: 保存済みの一覧JSONを読み込みます。

## ショートカット

| 操作 | ショートカット |
| --- | --- |
| 元に戻す | `Ctrl + Z` |
| やり直し | `Ctrl + Shift + Z` / `Ctrl + Y` |
| コピー | `Ctrl + C` |
| 貼り付け | `Ctrl + V` |
| 全選択 | `Ctrl + A` |
| 選択解除 | `Esc` |
| 削除 | `Delete` / `Backspace` |
| 0.1m移動 | 矢印キー |
| 1m移動 | `Shift + 矢印キー` |
| ズーム | `Ctrl + マウスホイール` |

Macでは `Ctrl` の代わりに `Command` も使えます。

## 集計の考え方

- **設置数**: 配置した荷物の個数です。柱・通路は含みません。
- **総収容数**: 荷物ごとの段積み数の合計です。
- **使用床面積**: 配置した荷物の床面積合計です。
- **有効床面積**: 倉庫の床面積から柱・通路などの障害物面積を引いた面積です。
- **空き有効床**: 有効床面積から使用床面積を引いた面積です。
- **床占有率**: 使用床面積 ÷ 有効床面積です。

## 注意点

- このアプリは単体HTMLで動作します。レイアウトを確実に残したい場合は、必ず **保存** または **一覧JSON保存** を実行してください。
- ブラウザや実行環境によっては、記録一覧の自動保存が使えない場合があります。その場合は一覧JSONで保存してください。
- 読込時は現在の画面内容が読み込んだJSONで置き換わります。必要な場合は先に保存してください。
- 荷物同士が重なった場合は赤枠で表示されます。配置を調整してください。
